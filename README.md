<h1>naqlGo</h1>
<h3><p>Une application web pour la gestion des expéditions. Construite avec TypeScript, Docker et une architecture frontend/backend moderne.
</p></h3>

<h2>✨ Fonctionnalités</h2>

<ul>
  <li>**Authentification utilisateur**</li>
  <li>**Gestion des expéditions**</li>
  <li>**Configuration conteneurisée**</li>
</ul>

<h2>🛠 Technologies utilisées</h2>

<ul>
  <li>**Backend** – TypeScript, Node.js (Express js)</li>
  <li>**Frontend** – TypeScript, nextjs, CSS (Tailwind)</li>
  <li>**Proxy** – Nginx</li>
  <li>**Conteneurisation** – Docker, Docker Compose</li>
</ul>

<h2>Prérequis</h2>

<ul>
  <li>[Node.js](https://nodejs.org/) version 24</li>
  <li>[Docker](https://docker.com/)</li>
  <li>[Docker Compose](https://docs.docker.com/compose/)</li>
</ul>

<h2>Installation & exécution</h2>
<h3>Backend</h3>
<ul>
  <li>git clone https://github.com/Hichamchaguer/naqlGo.git</li>
  <li>cd naqlGo</li>
  <li>docker-compose up --build</li>
</ul>

<h3>frontend</h3>

<ul>
  <li>cd frontend</li>
  <li>npm run dev</li>
</ul>

<h2>L'application sera disponible à l'adresse http://localhost (via Nginx)</h2>
